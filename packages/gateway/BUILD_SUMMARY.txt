HOW TO RUN
----------
1. pnpm install                    # Install deps
2. Copy .env.example to .env        # Configure secrets
3. Edit .env with real API keys     # Add KeeperHub, Uniswap keys
4. pnpm typecheck                  # Verify TypeScript
5. pnpm build                       # Compile to JS
6. pnpm dev                         # Start server (auto-reload)
   or
   pnpm start                       # Production server

ENVIRONMENT REQUIRED
--------------------
CCIP_SIGNER_KEY                   - 32-byte Ethereum private key (NO 0x prefix)
KEEPERHUB_API_KEY                 - KeeperHub API authentication
ZEROG_INDEXER_URL                 - 0G Storage indexer endpoint
AXL_NODE_URL                       - AXL consensus node for events
RPC_URL_0G                         - 0G Galileo RPC
(Optional) UNISWAP_API_KEY         - Uniswap trade API key
